Documentation

Size

width / height

  • Height is optional, if no height is specified the image will be a square. Example: http://placepuppy.it/300
  • Must be the first option in the url

Image Number

&n=14

  • You can request a specific image in the range 1-28.
  • Find the image number in the attribution page.
  • This is optional.
    • http://placepuppy.it/450/400&n=7

Custom Text

&text=Hello+World

  • Custom text can be entered using a query string at the very end of the url.
  • This is optional, if you don't specify anything you'll see only the image.
  • a-z (upper and lowercase), numbers, and most symbols will work just fine.
  • The vertical bar "|" is transformed in newline
  • Point size is calculated depending on the image size, minimum point size is 5.
  • If the image is too short text is truncated.
  • You can write spaces as + in the url, so doing math with placepuppy.it may present some difficulties ;-)

Custom Colors

&bg=yellow&fg=yellow

  • These options work only if text is present, if there is no text these options have no consequences.
  • You can specify a custom color for background (&bg) and text (foreground, &fg).
  • This is optional, if you don't specify anything you will obtain white text (&fg=FFFFFF) on dark grey (&bg=000008) background.
  • you can either use an hex code (without "#") to specify the color or the HTML color name.

Image Format

.jpg

  • All images on http://placepuppy.it/ are served as jpg images.

Flash Support

Since dummyimage.com works with Flash applications also placepuppy.it should.

  • Images are served with a specified Content-Length so they can be used in preloaders
  • placepuppy.it uses a crossdomain.xml file
